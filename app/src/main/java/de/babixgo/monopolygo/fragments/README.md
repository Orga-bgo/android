# Fragments Package

This package contains Fragment classes for modular UI components.

## Planned Classes

- **AccountEditFragment.java** - Fragment for editing account details
- **TeamEditFragment.java** - Fragment for editing team composition

## Purpose

Fragments provide reusable UI components that can be embedded in activities or dialogs, promoting code reuse and modularity.
