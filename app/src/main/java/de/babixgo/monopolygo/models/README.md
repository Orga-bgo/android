# Models Package

This package contains data model classes for the BabixGO application.

## Planned Classes

- **Account.java** - MonopolyGo account data model
- **Customer.java** - Customer information model
- **Event.java** - Tycoon Racers event model
- **Team.java** - Team composition model

## Purpose

Models represent the data structures used throughout the application, providing a clear contract between the database layer and the UI layer.
