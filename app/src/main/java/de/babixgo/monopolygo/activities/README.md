# Activities Package

This package will contain new Activity classes for the BabixGO application.

## Planned Classes

- **AccountListActivity.java** - Display and manage account list
- **AccountDetailActivity.java** - View and edit account details
- **EventListActivity.java** - Display and manage events
- **EventDetailActivity.java** - View and edit event details

## Existing Activities (in parent package)

- **MainActivity.java** - Main menu (EXISTING - will be extended)
- **AccountManagementActivity.java** - Account management (EXISTING - will be extended)
- **PartnerEventActivity.java** - Partner event management (EXISTING - will be extended)
- **FriendshipActivity.java** - Friendship management (EXISTING - will be extended)

## Purpose

Activities represent screens in the application. New activities will be placed here while existing activities remain in the parent package to maintain backwards compatibility.
