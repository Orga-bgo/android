# Adapters Package

This package contains RecyclerView adapters for list displays.

## Planned Classes

- **AccountListAdapter.java** - Adapter for account list display
- **EventListAdapter.java** - Adapter for event list display
- **TeamListAdapter.java** - Adapter for team composition display

## Purpose

Adapters bridge the gap between data models and RecyclerView UI components, handling the display and interaction logic for lists.
